## Setup
* First time setup (RUN TEST FIRST! EVERYTIME!!!)
  * Run the install.sh script inside the install directory.
* Create symlinks to these files for version controlled config files. e.g.
````
ln -s ~/DotFiles/.zshrc ~/
````
